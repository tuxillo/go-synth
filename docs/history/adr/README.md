# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting
significant architectural decisions made in the go-synth project.

## Format

Each ADR should follow this template:

```markdown
# ADR NNN: Title

**Status**: Proposed | Accepted | Deprecated | Superseded by [ADR-XXX]
**Date**: YYYY-MM-DD
**Deciders**: Names
**Context**: Brief context

## Decision

What was decided.

## Rationale

Why this decision was made.

## Consequences

- Positive consequences
- Negative consequences
- Neutral consequences

## Alternatives Considered

What other options were considered and why they were rejected.
```
